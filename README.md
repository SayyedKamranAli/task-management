# Task Management - Full Stack Application

A comprehensive task management system with role-based access control (Admin/Member). Built with MERN stack (MongoDB, Express.js, React, Node.js).

## ✨ Features

### Authentication & Authorization
- 🔐 JWT-based authentication
- 👥 Role-based access (Admin/Member)
- 📝 Signup/Login functionality
- 🔒 Protected routes

### Project Management
- 📁 Create, read, update, delete projects
- 👨‍💼 Admin can add/remove team members
- 👀 Members can only view assigned projects
- 📊 Project status tracking

### Task Management
- ✅ Create, assign, and track tasks
- 🏷️ Task priorities (Low, Medium, High)
- 📅 Due date management (DD/MM/YYYY format)
- 🔄 Task status updates (Pending, In Progress, Completed)
- 👥 Admin can assign tasks to any member
- 👤 Members can update their own task status

### Dashboard
- 📈 Task statistics overview
- ⏰ Overdue task alerts
- 📋 Recent tasks display
- 👁️ Role-based dashboard views

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI Library
- **React Bootstrap** - UI Components
- **React Router DOM** - Navigation
- **Axios** - API calls
- **React Icons** - Icons

### Backend
- **Node.js** - Runtime
- **Express.js** - Web Framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **bcryptjs** - Password hashing

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)
- npm or yarn
